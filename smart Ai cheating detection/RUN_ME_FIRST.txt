🚀 QUICK START (3 STEPS):

STEP 1: Install Dependencies
────────────────────────────
Open Command Prompt in this folder and run:

    pip install Flask mysql-connector-python opencv-python opencv-contrib-python mediapipe scikit-learn SpeechRecognition Pillow Werkzeug


STEP 2: Run Demo
────────────────
Test the system without database:

    python demo.py

Press 'q' to quit the demo.


STEP 3: (Optional) Setup Full System
─────────────────────────────────────
For complete functionality with database:

1. Install MySQL Server
2. Run: mysql -u root -p < schema.sql
3. Edit Database/db_connection.py (update password)
4. Run: python app.py
